
# Project Title

A simple React.js Project that generates startup company name.




## Run Locally

Clone the project

```bash
  git clone https://github.com/shakaib732/startupName.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
